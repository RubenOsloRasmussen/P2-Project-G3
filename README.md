# Sudoku website with difficulty control
Created by students from Aalborg University (AAU) <br />
Student group CS-26-SW-2-03<br />

## Overview
This project is a website where users can play sudoku puzzles with automatic difficulty adjustments.<br />
The website also includes a web page with strategies to solving puzzles. <br />
Speciffic puzzles are recommended to users depending on the difficulty of the sudoku puzzle.<br />

## Usage
Users will complete sudoku puzzles on the sudoku web page of the website. <br />
Completing the sudokus fast and without errors increases the difficulty of the next given sudoku, <br />
while taking a long time or making many errors will decrease the difficulty. <br />
While solving these puzzles, sudoku solving strategies will be recommended to the user. <br />
Learning these strategies and using them will increase the users skill in solving sudokus, <br />
allowing them to play mere difficult sudokus.
